🩺 Patient Records API

A REST API for managing patient medical records with full OpenAPI documentation.

📖 API Documentation 📋 OpenAPI Spec (JSON)

Endpoints

Method Path Description
GET /api/v1/patients List all patients
GET /api/v1/patients/{id} Get a patient by ID
POST /api/v1/patients Create a new patient
PUT /api/v1/patients/{id} Update an existing patient
DELETE /api/v1/patients/{id} Delete a patient

Patient Schema

{
  "id": "string (UUID v4, read-only)",
  "patient_id": "string",
  "patient_name": "string",
  "gender": "Male | Female | Other",
  "date_of_birth": "string (YYYY-MM-DD)",
  "visit_date": "string (YYYY-MM-DD)",
  "symptoms": "string",
  "diagnosis": "string",
  "medication": "string",
  "dosage": "string"
}
