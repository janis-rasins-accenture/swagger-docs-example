🩺 Patient Records API

A simple REST API for managing patient medical records.

Interactive API docs OpenAPI spec (JSON) List patients

Endpoints

Method Path Description
GET /api/v1/patients List all patients
GET /api/v1/patients/{id} Get a patient by ID
POST /api/v1/patients Create a new patient
PUT /api/v1/patients/{id} Update an existing patient
DELETE /api/v1/patients/{id} Delete a patient
